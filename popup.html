PopUp with Vue and Quasar

Нажать кнопку start по центру экрана.

Появляется модальное окно с полями, которые нужно заполнить.

Нажать далее, появляется второе модальное окно с данными, которые были в первом.

Нажать кнопку ок, окно исчезает и появляется кнопка start.

Scills: Html, CSS. Vue: v-modal, props, emit. Quasar
